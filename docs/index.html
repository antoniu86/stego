Encryption in
Plain Sight.

Hide files or folders inside images, videos, and PDFs without changing how they appear to the world.

Terminal — stego_cli
# Hide encrypted data
stego hide photo.jpg secret.txt -o hidden.jpg
# Extract data from carrier
stego show hidden.jpg -o ./recovered/
PBKDF2-HMAC-SHA256 initialized...

Invisible to Users

Carrier files remain fully functional and open in standard viewers.

High Security

100,000 iterations and SHA-256 checksums ensure data integrity.
