Hide. Encrypt.
Protect.

A lightweight tool to append encrypted data to any carrier file. Images, videos, and PDFs remain fully functional while hiding your secrets.

Active Session: stego-cli
$ stego hide photo.jpg data/ -o vault.jpg
> Enter encryption password...
$ stego scan vault.jpg
> Payload detected (AES-256)
$ stego show vault.jpg -o ./out/
> Extraction complete. SHA-256 verified.

AES-256 Security

PBKDF2-HMAC-SHA256 with 100,000 iterations for key derivation.

Universal Carriers

Works with any file format—JPG, MP4, PDF, and more.

Non-Destructive

The original carrier content is never modified or corrupted.
